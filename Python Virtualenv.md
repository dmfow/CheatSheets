#### Start a new environment
A directory will be created where you are in the library
```python
mkdir ~/code/py_vm
cd ~/code/py_vm
virtualenv thenewenvironment
```



## Install

#### Install with pip
```python
pip3 install virtualenv
```


#### Check the version
```python
virtualenv --version
```
